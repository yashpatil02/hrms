# HRMS System

This is a Human Resource Management System built using Node.js, Express, Prisma and React.

## Features
- Employee Management
- Document Upload
- Admin Panel
- Real-time Notifications

## Tech Stack
- Node.js
- Express
- Prisma
- React
- Socket.io

## Installation

1. Clone the repository
git clone https://github.com/yashpatil02/hrms.git

2. Install backend dependencies
npm install

3. Run the server
node server.js
